# Sodoku
Project title - Sudoku Board Game Project
Student name - Zahin Ahammad period 3 Ms. Qiu
Short description of the project - generates a sodoku board with random numbers using 2D arrays and arraylist.
How to run the program - Run the program and it would generate a board with random numbers for each square 1-9 and row and coulumn
Summary of how the Sudoku board is generated - I used an array list to help generate each row. It would fill in each number from the row. As it keeps on going it would use a loop to see if that the number is different from all the other numbers from the index row and column and if not it would keep generating a number until it gets it. Then it used the 2D array to fill in the number and also used it to print out the board.
